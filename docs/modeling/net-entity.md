# Net Entity Modeling

Net entities are virtual nodes that enforce power balance.

## Purpose

A net entity ensures:
\[ \sum_{\text{inflows}} P_{\text{in}}(t) = \sum_{\text{outflows}} P_{\text{out}}(t) \]

## Use Cases

- Hub for multiple connections
- Distribution points
- Complex network topologies

See [net entity configuration](../user-guide/entities/net.md).
